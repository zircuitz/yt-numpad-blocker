# YouTube Numpad Blocker

Google Chrome extension that Stops accidental numpad presses from skipping to a random point in a YouTube
video (YouTube maps Numpad0-9 to "seek to N0%").

Also supported in other chromium-based browsers (Brave, Edge, etc...)

### Telemetry and data collection.

This extension has *ZERO* telemetry and data collection built in.

I believe in your rights to privacy. There isn't even an option to turn it on. And I intend to keep it that way!
